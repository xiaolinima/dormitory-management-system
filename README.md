# 学生宿舍管理系统 | Система управления студенческими общежитиями

---

## 📋 项目简介 | Описание проекта

一个功能完整的学生宿舍管理系统，支持学生和管理员两种角色，实现从宿舍申请、床位分配、维修服务到费用管理的全流程业务。 | Полнофункциональная система управления студенческими общежитиями с поддержкой двух ролей: студента и администратора, реализующая полный бизнес-процесс от подачи заявки на общежитие, распределения мест, технического обслуживания до управления платежами.

## 🚀 快速开始 | Быстрый старт

### 前置条件 | Предварительные требования
- Node.js 18+
- Java 21+
- PostgreSQL 15+
- Maven 3.8+

### 后端启动 | Запуск бэкенда
```bash
cd xbqbackend
mvn spring-boot:run
```
后端服务运行在 http://localhost:8080 | Бэкенд работает на http://localhost:8080

### 前端启动 | Запуск фронтенда
```bash
cd xbqfrontend
npm install
npm run dev
```
前端服务运行在 http://localhost:5173 | Фронтенд работает на http://localhost:5173

### 数据库配置 | Настройка базы данных
1. 创建数据库 `xbq` | Создать БД `xbq`
2. 执行 `xbq_all_in_one.sql` 初始化表 | Выполнить `xbq_all_in_one.sql` для инициализации таблиц
3. 可选：执行 `create_application_table.sql` 添加宿舍申请功能 | Опционально: выполнить `create_application_table.sql` для добавления функции подачи заявок

## 🎯 核心业务流程 | Основной бизнес-процесс

1. 学生申请宿舍 | Студент подает заявку на общежитие
2. 管理员审批申请 | Администратор рассматривает заявку
3. 系统分配床位 | Система распределяет место
4. 学生查看分配结果 | Студент просматривает результат
